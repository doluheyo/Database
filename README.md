## 檔案與資料夾說明
- templates 放置所有在瀏覽器 (前端) 看得到的東西
	- 包含: html、css、js 檔案以及會用到的圖片，都放在這個資料夾
- ```app.py``` 是伺服器 (後端) 的主要程式，有了這個前端才能跟後端的資料庫互動
	- 它的架構按照順序就是
		- 與資料庫建立連線
		- 寫各種網站要使用到資料庫的功能 (API)
		- 設定實際要在前端使用的 html 檔案
		- 啟動以上的程式碼

## 目前的執行流程
- 在一台電腦中，建立虛擬機
	- 將虛擬機作為後端的伺服器 (Server)，負責執行 ```app.py``` 並連接資料庫
	- 用原本的電腦作為前端的客戶端 (Client)，負責訪問網址，扮演使用者進行操作
	- 因為都在同一台電腦內，算是走內網(?)，就沒有什麼連接不到伺服器的問題，甚至電腦本身沒有接網路也是可以運行的
	- 在客戶端要連接之前，需要先在虛擬機執行檔案 ```python3 app.py```，客戶端再去訪問虛擬機的 ip 與 設定的 port 號
- 目前先這樣去假裝老師要的 Client-Server 架構，等系上的機器到手了再來看要怎麼修改
- 但 ```app.py``` 的功能跟那些網頁，還有資料庫的資料都應該可以先準備，這些應該不會被影響
